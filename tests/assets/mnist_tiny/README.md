# MNIST Tiny Fixture

This directory contains a deterministic subset of MNIST for unit tests.

## Contents

- `train_images.npy`: Training images (uint8, shape [64, 28, 28])
- `train_labels.npy`: Training labels (int64, shape [64])
- `test_images.npy`: Test images (uint8, shape [64, 28, 28])
- `test_labels.npy`: Test labels (int64, shape [64])

## Generation

Generated by `scripts/make_mnist_tiny.py` with:
- K = 64 samples per split
- Indices: first 64 samples from each split (deterministic)
- Source: torchvision.datasets.MNIST

## Label Distribution

Train labels: {np.int64(0): np.int64(7), np.int64(1): np.int64(8), np.int64(2): np.int64(4), np.int64(3): np.int64(8), np.int64(4): np.int64(8), np.int64(5): np.int64(4), np.int64(6): np.int64(6), np.int64(7): np.int64(5), np.int64(8): np.int64(5), np.int64(9): np.int64(9)}
Test labels: {np.int64(0): np.int64(6), np.int64(1): np.int64(10), np.int64(2): np.int64(5), np.int64(3): np.int64(6), np.int64(4): np.int64(10), np.int64(5): np.int64(7), np.int64(6): np.int64(5), np.int64(7): np.int64(7), np.int64(8): np.int64(1), np.int64(9): np.int64(7)}

## Usage
